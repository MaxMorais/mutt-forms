    MuttConfig

    Internal registry for Mutt fields & widgets. This is used internally to register default fields & widgets, and can also be used as a hook to install new fields & widgets via plugins.

    Constructor

    new MuttConfig()

    MuttConfig is used for configuration information. It's primarily used by the builder utility to decide field mappings and by the plugins to extend the core functionality.
